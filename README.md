Eisen Planner

Eisen Planner is a responsive, user-friendly productivity tool inspired by the Eisenhower Matrix. 
Designed with a clean pastel aesthetic and intuitive interface, this planner helps users manage daily, 
weekly, and monthly tasks with clarity and efficiency.

# Features

- ✅ Daily task management using the Eisenhower Matrix (Urgent/Important grid)
- 📆 Weekly and Monthly calendar views with task scheduling
- 📝 Drag-and-drop functionality for organizing tasks between sections
- 🔄 Undo delete functionality for recently removed tasks
- 🌙 Light and dark theme toggling
- ✅ Checklist-style tasks with completion tracking
- 💬 Daily motivational quotes for user inspiration
- 💻 Fully client-side — no backend required

# Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

# How to Use

1. Clone or download the repository.
2. Open `index.html` in your preferred web browser.
3. Add tasks through the input fields in Daily, Weekly, or Monthly tabs.
4. Drag, drop, complete, or delete tasks as needed.
5. Switch themes and enjoy a streamlined productivity experience.
